# 🌸 Ax_beauty — Guide de déploiement

## Étape 1 — Créer un compte GitHub (gratuit)
1. Allez sur **github.com**
2. Cliquez **Sign up** → créez un compte gratuit

## Étape 2 — Uploader le projet sur GitHub
1. Connecté sur GitHub, cliquez **"+"** en haut → **"New repository"**
2. Nom : `axbeauty` → cliquez **"Create repository"**
3. Sur la page suivante, cliquez **"uploading an existing file"**
4. Glissez-déposez **tous les fichiers** de ce dossier
5. Cliquez **"Commit changes"**

## Étape 3 — Déployer sur Vercel (gratuit)
1. Allez sur **vercel.com**
2. Cliquez **"Sign up"** → choisissez **"Continue with GitHub"**
3. Cliquez **"Add New Project"**
4. Sélectionnez votre repo **axbeauty**
5. Vercel détecte automatiquement Vite/React ✓
6. Cliquez **"Deploy"** → attendez 2 minutes
7. Vous obtenez votre URL : `axbeauty.vercel.app` 🎉

## Étape 4 — Installer sur l'écran d'accueil iPhone
1. Ouvrez votre URL dans **Safari** (pas Chrome !)
2. Appuyez sur l'icône **Partager** (carré avec flèche vers le haut)
3. Faites défiler → appuyez **"Sur l'écran d'accueil"**
4. Nommez-la **"Ax_beauty"** → appuyez **"Ajouter"**
5. L'app apparaît sur votre écran comme une vraie app ! ✓

---
⚠️  Important : utilisez toujours **Safari** sur iPhone pour installer.
Chrome ne supporte pas cette fonctionnalité sur iOS.
